Game Programming con Game Maker Studio

76 ore di lezione stimate · avanzato

A chi è destinato il corso

Il corso su Game Maker Studio 1.4 è destinato a tutti i programmatori con esperienza pregressa con linguaggi che non siano Object Oriented.
Il corso è considerato avanzato siccome, a causa del linguaggio datato, molti aspetti e funzioni del linguaggio stesso sono decisamente ostici, quindi è bene avere un minimo di esperienza con qualche linguaggio almeno lontanamente simile (magari C o JavaScript).
Inoltre, a differenza di altri engine più recenti, la gestione di risorse e script è molto più ostica e necessita maggiore pratica e attenzione.

Requisiti per il corso

  • Avere esperienza pregressa con linguaggi non Object Oriented, possibilmente C e JavaScript
  • Possedere una licenza di Game Maker Studio 1.4
