Game Programming con Unity

96 ore di lezione stimate · intermedio

A chi è destinato il corso

Unity è uno degli engine più utilizzati al mondo e permette di sviluppare complessi videogiochi tramite un potente linguaggio ad oggetti quale è C#.
Il corso è dedicato a tutti coloro che vogliono iniziare ad approcciarsi al mondo dello sviluppo videoludico: non è richiesta alcuna esperienza su Unity o su qualsiasi altro engine, ma è fortemente richiesta esperienza con almeno un linguaggio di programmazione ad oggetti (meglio se Java) o comunque la conoscenza della teoria Object Oriented. Questo perchè C# è un linguaggio Object Oriented e, per lavorare su Unity, è strettamente necessario conoscere determinati paradigmi.
È, infine, importante chiarire che questo corso, come da titolo, si concentra sulla programmazione e sul funzionamento generale di Unity e degli engine in generale, quindi non è un corso sul Game Design.

Requisiti per il corso

  • Conoscere la teoria dei linguaggi Object Oriented o conoscere almeno un linguaggio Object Oriented
