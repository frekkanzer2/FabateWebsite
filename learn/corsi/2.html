Game Programming con Unity

Unity è uno degli engine più famosi nel campo dello sviluppo videoludico siccome è molto semplice e potente. Permette di sviluppare su qualsiasi piattaforma, quindi PC, mobile e console.
Il corso riguarderà la creazione dei videogiochi e tutti gli aspetti che la circondano, quindi non ci si concentrerà solamente sulla programmazione.
Otterrai tutte le basi necessarie per progettare, sviluppare videogiochi e per iniziare la tua carriera nell'ambito videoludico.

96 ore di lezione · intermedio

Game Programming con Game Maker Studio

Game Maker Studio 1.4 è un famoso engine a pagamento utile alla creazione di videogiochi 2D in maniera rapida e, per certi versi, più semplice rispetto a Unity.
Essendo anche un engine relativamente datato, soffre di alcune pecche ma rimane comunque un valido engine di tutto rispetto.
Game Maker Studio offre un sistema di drag-and-drop che non vedremo nel corso siccome ci dedicheremo direttamente al linguaggio dell'engine, il GML, il quale vi ricorderà per certi versi il C e JavaScript non essendo Object Oriented.

76 ore di lezione · avanzato

Sviluppo multipiattaforma con Flutter

Flutter è un framework open-source creato da Google inizialmente utile per la creazione di interfacce grafiche per iOS e Android.
Con il tempo, Flutter si è evoluto e attualmente è uno dei framework preferiti dagli sviluppatori siccome permette la creazione di applicazioni complete (e sì, non solo interfacce) sia per iOS, che per Android, che per Windows!
Flutter permette di realizzare in maniera semplice e veloce le interfacce grafiche, mentre Dart si occupa del comportamento dell'applicativo. Niente panico: Dart è il linguaggio dal quale Flutter è nato.
Vedremo, in questo corso, come utilizzare Flutter/Dart e come realizzare applicativi multipiattaforma con pochissimi click!

20 ore di lezione · intermedio

Dettagli del corso

Database relazionali (SQL)

I database relazionali sono grosse collezioni che permettono di salvare dati in relazione tra loro.
I relazionali sono la tipologia di database utilizzata in molte applicazioni: ecommerce, social network, programmi aziendali. Insomma, tutti quei programmi in cui può servire creare relazioni tra tipologie di dati.
Questo corso offre tutte le basi per comprendere il funzionamento dei database relazionali e come strutturarli, aspetto molto importante siccome da questo scindono le prestazioni del sistema nel complesso e tanti altri aspetti.
Conoscere i database relazionali, per un programmatore, è un must-have!
