Le basi dello sviluppo web

70 ore di lezione stimate · facile

A chi è destinato il corso

Questo è il corso ideale per chi si approccia per la prima volta al mondo dello sviluppo web.
Il corso inizia dando l'infarinatura giusta per iniziare a sviluppare qualche semplice sito web. Ovviamente non è il corso ideale nel caso vogliate imparare a sviluppare Web Application complesse con tanto di backend: per quello servono tante altre nozioni avanzate. Perciò, è bene iniziare con questo corso per poi iniziare a lavorare, in successiva, su applicativi web avanzati.
